Single Flavor Mode

Only dark/light toggle - no flavor selector in the toolbar

With one flavor, pressing T does nothing. Only D toggles dark/light.

This is the default build mode

When you run sldr build my-talk --flavor acme, only that one flavor is embedded. The output is smaller and the toolbar is simpler.

sldr build my-talk --flavor acme       # single flavor (default)
sldr build my-talk --flavors acme,dark  # multi-flavor, selector shown

Done

Press D to toggle dark mode
